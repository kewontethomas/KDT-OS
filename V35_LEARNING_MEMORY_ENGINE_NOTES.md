# KDT OS V35 Learning Memory Engine

V35 turns verified proof into long-term skill memory.

## Adds
- `kdt_learning_memory_v35.py`
- `templates/learning_memory.html`
- `/learning_memory`
- `/learning_memory.json`
- `reports/learning_memory.json`
- `reports/skill_timeline.json`

## Purpose
KDT OS can now answer:
- What has Kewonte actually proven?
- When was a skill first verified?
- When was it last verified?
- Which skills are strong?
- Which skills need refresh?

## Trust Rule
Estimated skill is not the same as remembered proof. V35 only treats a skill as strong memory when it has promoted proof history.
