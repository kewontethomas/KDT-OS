# Autonomous Growth Update

Growth is no longer built around one-off repair buttons.

Permanent Growth actions:
- Run Visual Self-Inspection
- Review Repair Queue
- Add Screenshot Evidence
- Run Basic Health Check

Flow:
1. Run Visual Self-Inspection.
2. FieldSeed detects issues.
3. FieldSeed creates repair candidates automatically.
4. You open Review Repair Queue.
5. You approve the repair.
6. FieldSeed backs up files, applies repair, compile-tests, and rolls back if needed.
