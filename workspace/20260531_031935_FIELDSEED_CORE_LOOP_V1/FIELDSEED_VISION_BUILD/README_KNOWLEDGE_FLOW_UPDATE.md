# FieldSeed Knowledge Flow Update

Yes, the database updates itself as you use the app.

What changed:
- Natural knowledge understanding layer
- Open Tickets page
- Select open tickets by ID
- Add findings to selected ticket
- Mark tickets Waiting / In Progress
- Close selected tickets with confirmed root cause + resolution
- Closing a ticket automatically creates confirmed knowledge
- Brain responses are more natural and still truth-locked

Important:
FieldSeed still will not guess. If the knowledge base does not contain a confirmed match, Brain says it does not know yet and asks you to add evidence/findings.
