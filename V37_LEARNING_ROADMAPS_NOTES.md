# KDT OS V37 - Learning Roadmaps

V37 adds roadmap intelligence so KDT OS can answer what comes next after a skill, not just what skill is currently weak.

## Added
- `kdt_learning_roadmaps_v37.py`
- `templates/learning_roadmaps.html`
- `/learning_roadmaps`
- `/learning_roadmaps.json`
- `reports/learning_roadmaps.json`
- `reports/roadmap_progress.json`

## Purpose
KDT OS now maps skills and proof into ordered growth paths such as Python Developer, Web App Builder, Flask Backend, SQLite/Data, IT Support, and KDT OS Builder.

## Rule
Roadmaps should guide direction. They should not claim mastery unless proof or promotion exists.
